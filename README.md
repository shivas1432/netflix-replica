## 🚀 Setup

```bash
# Clone & Install
git clone https://github.com/shivas1432/netflix-replica.git
cd netflix-replica
npm install

# Environment Setup
echo "REACT_APP_API_KEY=your_tmdb_key" > .env
echo "REACT_APP_FIREBASE_API_KEY=your_firebase_key" >> .env

# Run
npm start
```

**API Keys:** [TMDB API](https://www.themoviedb.org/settings/api) • [Firebase Console](https://console.firebase.google.com/)
